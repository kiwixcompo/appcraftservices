## Troubleshooting

### "Authentication failed" error:
1. Make sure you're logged into GitHub in your browser
2. You might need to set up a Personal Access Token
3. Go to GitHub Settings > Developer settings > Personal access tokens
4. Create a token with repository permissions

### "No internet connection" error:
- Check your internet connection
- Try again when connected
- Your changes are saved locally and will sync when connection is restored

### Files not syncing:
- Make sure you're running the batch file from the correct folder
- Check that the files you want to sync aren't listed in `.gitignore`

## Need Help?
- Check the messages in the command window for specific errors
- All batch files include helpful error messages
- Your changes are always saved locally first, so nothing is lost
