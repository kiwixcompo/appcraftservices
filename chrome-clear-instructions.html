Clear Chrome Data for appcraftservices.com

Follow these steps exactly to clear Chrome's security cache:

Step 1: Clear Site Data

  1. In Chrome, press F12 to open DevTools
  2. Go to Application tab
  3. In the left sidebar, click Storage
  4. Click "Clear site data" button
  5. Check ALL boxes and click "Clear site data"

Step 2: Clear Browsing Data

  1. Press Ctrl+Shift+Delete (or Cmd+Shift+Delete on Mac)
  2. Select "All time" from the time range
  3. Check these boxes:
    • ✓ Browsing history
    • ✓ Cookies and other site data
    • ✓ Cached images and files
  4. Click "Clear data"

Step 3: Reset Site Settings

  1. Type in address bar: chrome://settings/content/all
  2. Search for: appcraftservices.com
  3. Click the trash icon next to any entries for this site
  4. Close and restart Chrome completely

Step 4: Test

  1. Open a new Chrome window
  2. Visit: https://appcraftservices.com
  3. If still blocked, try: http://appcraftservices.com
